WHAT THE PROJECT DOES
  This projectis built to help track applications and where in the process they are at while you are job searching
WHY I BUILT IT
  I built this because it was hard to keep up with every job application and website I had to visit to make an application. This lets me keep track of all the jobs and if I have applied for the position already.
HOW TO RUN IT
  If you download both .py files you will be able to run jobtracker.py and see how it works
WHAT I LEARNED
  I leaned how to use SQLite and make a functioning user interface and keep data accessible when you re-run the program.
