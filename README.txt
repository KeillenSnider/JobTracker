WHAT THE PROJECT DOES: 
  A simple job tracking application to help users manage their job search process.You are able to keep track of a company name, role of the application, location of the business,
status of the application, notes you want to keep in mind about the company, and a URL because many different job applications have different places to see what stage your application is in.
This helps you keep track of the websites and also to know were you have applied before.

WHY I BUILT IT: 
  I built this because it was hard to keep up with every job application and website I had to visit to make an application. This lets me keep track of all the jobs and if I have applied for the position already.

HOW TO RUN IT: 
  If you download both database.py and jobtracker.py files you will be able to run jobtracker.py and see how it works in the terminal
OR you can use the website with the try now button on my main page

WHAT I LEARNED: 
  I leaned how to use SQLite and psycopg2 to make databases so I can keep data of users that will last over session and will not go away.
I learned how to use SQL to communicate with the databases and also HTML to make the websites that have working links and take information,
and send it to the app file to make changes to the database and what shows up on the screen. I also leaned how to use CSS to make a website 
not look bland and have personality.
